Offline storage Features

Offline Storage offers a powerful, local database and performance-optimized query engine that enables fast, reliable offline experiences across mobile platforms.

  • Local Filesystem Storage

    Data is stored directly on the filesystem outside of the browser environment.

  • SQL Query Engine

    A powerful query engine for building powerful, data-driven apps with SQL.

  • On-Device Data Encryption

    Data is secured on the device using 256-bit AES full database encryption.

  • Cross-Platform Deployment

    Works on supported iOS and Android mobile devices, from a single codebase.

  • Transaction Protection

    Implements serializable transactions that are atomic, consistent, isolated, and durable.

  • Zero Configuration

    Smart defaults that work out-of-the-box. You don’t need to be an expert to use it.
